2058. Структуры данных. Количество подстрок
ограничение времени на тест: 2 seconds
ограничение памяти на тест: 64 megabytes
ввод: standard
вывод: standard

Дана строка s, состоящая только из строчных латинских букв. Подстрокой называется некоторая непустая связная часть строки.

Ваша задача — подсчитать количество различных подстрок в этой строке.


Входные данные
Входные данные состоят из единственной строки s, длиной не более 500 символов.


Выходные данные
Выведите единственное целое число — количество различных подстрок данной строки.


Пример(ы)
input.txt
output.txt
aab
5

input.txt
output.txt
aabcd
14

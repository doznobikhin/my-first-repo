2052. Структуры данных. Вычеркиваем числа
ограничение времени на тест: 0.75 seconds
ограничение памяти на тест: 64 megabytes
ввод: standard
вывод: standard



Дана последовательность из n чисел a1, a2,..., an. C ней последовательно производят k операций: на i-ом шаге из этой
 последовательности одновременно вычеркивают все числа, стоящие на позициях с номерами, кратными bi (позиции нумеруются
 с 1).

Ваша задача — промоделировать это процесс и вывести массив после выполения всех операций.


Входные данные
Первая строка входного файла содержит числа n и k — количество элементов в последовательности и количество операций
 (1 ≤ n ≤ 105, 1 ≤ k ≤ 10). Во второй строке находится последовательность a1, a2,..., an (1 ≤ ai ≤ 109). В третьей
 строке находится последовательность b1, b2,..., bk (1 ≤ bi ≤ 105).


Выходные данные
Выведите последовательность после применения всех операций.


Пример(ы)
input.txt
7 2
1 2 3 4 5 6 7
2 3
output.txt
1 3 7

input.txt
4 1
5 4 9 3
3
output.txt
5 4 3
